## Navigation

#### Map

| Input             | Action            |
|:-------------     |:------------------|
| LMB               | Pan the map       |
| RMB               | Open Context Menu |
| MMB               | Select Item       |
| CTRL+LMB          | Distance Measure  |
| ESC               | Clear Selection   |

#### Tables

| Input             | Action            |
|:-------------     |:------------------|
| LMB               | Select Item       |
| CTRL+LMB          | Open Stream (Streamer Table) |
